3. FM Synthesis Engine

Core FM Principles

Frequency Modulation

One oscillator (modulator) varies the frequency of another (carrier) at audio rates, creating complex harmonic content from simple sine waves.

Operators

Each of the 6 operators can function as either a carrier (sound output) or modulator (frequency control), determined by the selected algorithm.

Modulation Index

The depth of frequency modulation, controlled by the modulator's output level and envelope, determines the harmonic complexity.

Feedback

Operator 1 can modulate itself, adding harmonic distortion and creating more complex waveforms from simple sine waves.

4. Algorithm Implementation

The DX7's 32 algorithms define how the 6 operators connect to each other. Each algorithm specifies which operators act as carriers (producing sound) and which act as modulators (controlling other operators' frequencies).

Algorithm Categories

Algorithms 1-8

Complex FM Chains: Long modulation chains for rich, evolving timbres. Perfect for brass, bells, and complex leads.

Algorithms 9-16

Mixed Configurations: Combination of chains and parallel operators. Ideal for electric pianos and hybrid sounds.

Algorithms 17-24

Parallel Operations: Multiple independent carriers with fewer modulators. Great for organ sounds and pads.

Algorithms 25-32

Additive Synthesis: All or most operators as carriers. Perfect for organ tones and harmonic stacks.

6. Envelope System

Envelope Stages

Stage Rate (0-99) Level (0-99) Behavior
Attack R1 - Speed to L1 L1 - Peak level Rise from 0 to peak when key pressed
Decay 1 R2 - Speed to L2 L2 - First sustain Fall from peak to first plateau
Decay 2 R3 - Speed to L3 L3 - Second sustain Slow decay while key held
Release R4 - Speed to L4 L4 - End level (usually 0) Fall to zero when key released

Rate Scaling

DX7 Rate Table: Envelope rates follow an exponential curve where rate 99 = ~0.4ms and rate 0 = ~30 seconds. Higher notes can have faster envelopes when key rate scaling is enabled, simulating acoustic instrument behavior.

7. Perfect Loop System

Zero-Crossing Loop Technology: Advanced algorithm ensures perfect loops by detecting LFO cycles and finding exact zero crossings for seamless audio loops.

Loop Generation Process

1. Find Start Zero

Algorithm searches for the first zero crossing after synthesis begins, ensuring clean loop start point.

2. Count LFO Cycles

Tracks LFO phase changes to count complete modulation cycles for musical synchronization.

3. Find End Zero

After target cycles complete, searches for next zero crossing to ensure seamless loop end.

4. Perfect Alignment

Forces exact 0.0 values at start/end points and shifts loop to begin at sample 0.

Command Line Interface
